Water Only Shader

A really simple Optifine shaders Minecraft: Java edition







Features

  • Screen space Reflection & Refraction
  • Water absorption with vanilla Minecraft biome colors
  • Water waving (simple waving in vertex and trochoidal waving in fragment)

Note !

This shader is only for Minecraft with Optifine mod, Iris mod support is planned for the next update.
Support for Minecraft 1.12.. - 1.18..
