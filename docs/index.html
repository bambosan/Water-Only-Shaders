Water Only Shader

A really simple Minecraft: Java edition shader







Features

  • Screen Space Reflection
  • Screen Space Refraction
  • Water Absorption
  • Water Waving (simple waving in vertex and trochoidal waving in fragment)

Note !

This shader is only for Minecraft with Optifine mod, Iris mod support is planned for the next update
